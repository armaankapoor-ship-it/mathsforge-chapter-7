01Basics

Indefinite integral

An indefinite integral gives all anti-derivatives of a function.

  • CBSE writing: Always add +C for indefinite integrals.
  • JEE focus: Check answer by differentiating quickly.
  • Common trap: Missing +C loses marks in board-style solutions.
02Power

Power rule

Increase the power by 1 and divide by the new power, except for x^-1.

  • CBSE writing: State n not equal to -1 when using the rule.
  • JEE focus: Separate x^-1 as log form.
  • Common trap: Integral of 1/x is log|x|, not x^0/0.
03Substitution

Substitution method

If the derivative of an inner expression is present, set that expression as u.

  • CBSE writing: Write u and du clearly.
  • JEE focus: Substitution is often hidden by constants.
  • Common trap: Do not forget to transform dx.
04Parts

Integration by parts

For products, integrate one factor and differentiate the other.

  • CBSE writing: Use ILATE to choose the first function.
  • JEE focus: Repeated parts solves log, inverse trig, and exponential products.
  • Common trap: Wrong first function creates longer work.
05Partial

Partial fractions

Rational functions with factorable denominator can be decomposed into simpler fractions.

  • CBSE writing: Do long division first when degree numerator is not smaller.
  • JEE focus: Repeated and irreducible factors need special forms.
  • Common trap: Skipping proper fraction check causes wrong setup.
06Definite

Definite integral

A definite integral gives a number, usually signed area.

  • CBSE writing: Apply upper limit minus lower limit after finding anti-derivative.
  • JEE focus: Use properties before integrating.
  • Common trap: No +C is needed in definite integrals.
07Property

Symmetry properties

Even and odd functions over symmetric intervals simplify strongly.

  • CBSE writing: Identify even or odd before evaluating.
  • JEE focus: Symmetry can turn hard integrals into zero or double area.
  • Common trap: Symmetric interval must be [-a,a].
08Property

Definite integral reversal

Changing the order of limits changes the sign.

  • CBSE writing: Use this to correct limit order.
  • JEE focus: Common in transformation questions.
  • Common trap: Integral from b to a equals negative integral from a to b.

Formula sheet

High-yield formulas with exact usage conditions.

Use these as quick revision cards before solving examples.

Power

Power rule

integral x^n dx = x^(n+1)/(n+1) + C, n != -1

Basic algebraic integral.

  • Example: integral x^4 dx = x^5/5 + C.
  • Trap: Does not work for n=-1.
Log

Log form

integral 1/x dx = log|x| + C

Special case of x^-1.

  • Example: integral 3/x dx = 3log|x|+C.
  • Trap: Absolute value is important.
Trig

Basic trig integrals

integral sin x dx = -cos x + C, integral cos x dx = sin x + C

Reverse trig derivatives.

  • Example: integral 5cos x dx = 5sin x + C.
  • Trap: Sign of integral sin x is negative cos x.
Inverse

Inverse trig form

integral dx/(1+x^2) = tan^-1 x + C

Arctan standard form.

  • Example: integral dx/(1+4x^2) = (1/2)tan^-1(2x)+C.
  • Trap: Adjust coefficient inside square.
Parts

Integration by parts

integral u v dx = u integral v dx - integral(u prime integral v dx) dx

Product integration rule.

  • Example: integral x e^x dx = x e^x - e^x + C.
  • Trap: Choose u carefully.
Definite

Limit reversal

integral_a^b f(x)dx = - integral_b^a f(x)dx

Changing limits changes sign.

  • Example: integral_2^0 x dx = -2.
  • Trap: Do not ignore order.
Property

Interval split

integral_a^b f = integral_a^c f + integral_c^b f

Split interval at any c.

  • Example: Use to handle modulus functions.
  • Trap: Keep limits in order.
Symmetry

Odd even shortcut

integral_-a^a odd = 0, integral_-a^a even = 2 integral_0^a f

Symmetry property.

  • Example: integral_-2^2 x^3 dx = 0.
  • Trap: Only on symmetric limits.

Derivation bank

Proofs that can be reproduced in exams.

Open a proof, follow the steps, and check the final result.

Derive the power rule
  1. Differentiate x^(n+1)/(n+1).
  2. The derivative is x^n.
  3. Therefore it is an anti-derivative of x^n.
  4. Add +C for the family of anti-derivatives.

Final: integral x^n dx = x^(n+1)/(n+1)+C.

Shortcut: Reverse the derivative of x^(n+1).

Warning: n must not be -1.

Derive integration by parts
  1. Use d(uV)=u dV + V du.
  2. Let dV = v dx.
  3. Rearrange u v dx = d(uV) - V du.
  4. Integrate both sides.

Final: integral u v dx = uV - integral V du.

Shortcut: Start from product rule.

Warning: V is integral of v.

Prove limit reversal
  1. Let F be an anti-derivative of f.
  2. integral_a^b f = F(b)-F(a).
  3. integral_b^a f = F(a)-F(b).
  4. Hence the two are negatives.

Final: integral_a^b f = - integral_b^a f.

Shortcut: Use F(b)-F(a).

Warning: Same anti-derivative must be used.

Odd function definite property
  1. Split integral from -a to 0 and 0 to a.
  2. Use substitution x=-t in the first part.
  3. For odd f, f(-t)=-f(t).
  4. Two parts cancel.

Final: integral_-a^a odd function = 0.

Shortcut: Pair x and -x.

Warning: Limits must be symmetric.

Exam-style questions with answer reveal.

CBSEEasy

1. Find integral x^3 dx.

Answer: x^4/4 + C

Use power rule with n=3.

Trap: Do not forget +C.

CBSEEasy

2. Find integral 1/x dx.

Answer: log|x| + C

This is the n=-1 special case.

Trap: Do not use power rule denominator zero.

TrigMedium

3. Find integral sin x dx.

Answer: -cos x + C

Derivative of -cos x is sin x.

Trap: Sign is often missed.

DefiniteMedium

4. Evaluate integral_0^2 x dx.

Answer: 2

Anti-derivative x^2/2 gives 4/2-0=2.

Trap: No +C in final definite answer.

PropertyMedium

5. Evaluate integral_-1^1 x^5 dx.

Answer: 0

x^5 is odd and limits are symmetric.

Trap: Symmetry must be checked.

PartsMedium

6. Find integral x e^x dx.

Answer: e^x(x-1)+C

Use parts with u=x and dv=e^x dx.

Trap: Choose u=x by ILATE.

SubstitutionMedium

7. Find integral 2x cos(x^2) dx.

Answer: sin(x^2)+C

Let u=x^2, du=2x dx.

Trap: Inner derivative must appear.

InverseMedium

8. Find integral dx/(1+x^2).

Answer: tan^-1 x + C

Use standard arctan form.

Trap: Do not write log form.

PartialHard

9. What should be done before partial fractions if numerator degree >= denominator degree?

Answer: Divide first

Make the fraction proper by polynomial division.

Trap: Partial fractions setup requires proper rational expression.

PropertyHard

10. Write integral_a^b f(x)dx in terms of integral_b^a.

Answer: - integral_b^a f(x)dx

Reversing limits changes sign.

Trap: Magnitude same, sign opposite.

CBSEHard

11. Find integral log x dx.

Answer: x log x - x + C

Use parts with u=log x and dv=dx.

Trap: Cannot integrate log x by simple power rule.

TrapMedium

12. Is integral of f prime(x) dx equal to f(x)+C?

Answer: Yes

Integration reverses differentiation.

Trap: Constant is necessary for indefinite integrals.

Core formulas

  • Power rule: integral x^n dx = x^(n+1)/(n+1) + C, n != -1
  • Log form: integral 1/x dx = log|x| + C
  • Basic trig integrals: integral sin x dx = -cos x + C, integral cos x dx = sin x + C
  • Inverse trig form: integral dx/(1+x^2) = tan^-1 x + C
  • Integration by parts: integral u v dx = u integral v dx - integral(u prime integral v dx) dx
  • Limit reversal: integral_a^b f(x)dx = - integral_b^a f(x)dx

Common traps

  • Add +C for indefinite integrals.
  • No +C is required after a definite integral value.
  • integral 1/x dx is log|x|+C.
  • Substitution must transform dx.
  • Integration by parts depends on first-function choice.
  • Symmetry shortcuts need symmetric limits.
